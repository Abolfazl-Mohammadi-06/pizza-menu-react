# 🍕 Fast React Pizza Co.

A simple React practice project that simulates a pizza restaurant menu.

This project was built as a learning exercise to practice core React concepts such as component architecture, props, list rendering, conditional rendering, and dynamic styling.

## 🚀 Features

- Display a list of pizzas from a data source
- Render components dynamically using `map()`
- Pass data between components with Props
- Conditional rendering for sold-out items
- Dynamic CSS classes based on state
- Responsive layout for mobile and desktop devices

## 🛠️ Built With

- React
- JavaScript (ES6+)
- CSS3

## 📚 React Concepts Practiced

- Functional Components
- Props
- Component Composition
- Conditional Rendering
- List Rendering
- Dynamic Styling
- Reusable UI Components

## 📸 Preview

A restaurant menu application that displays available pizzas, ingredients, prices, and sold-out status.

## 🎯 Purpose

The main goal of this project is to strengthen React fundamentals through a small real-world UI example.

## ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/Abolfazl-Mohammadi-06/fast-react-pizza-co.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```
